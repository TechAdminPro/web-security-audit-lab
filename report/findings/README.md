# Hallazgos de la auditoría

En esta sección se presentan las vulnerabilidades identificadas durante la auditoría de seguridad realizada sobre la aplicación DVWA.

Cada hallazgo ha sido documentado de forma individual, incluyendo:

- Clasificación según OWASP  
- Nivel de severidad  
- Descripción técnica  
- Evidencias  
- Impacto  
- Recomendaciones  

## Lista de vulnerabilidades

1.  [Brute Force](./01_bruteforce.md)  
2.  [Command Injection](./02_command_injection.md)  
3.  [File Inclusion](./03_file_inclusion.md)  
4.  [File Upload](./04_file_upload.md)  
5.  [SQL Injection (Blind)](./05_sql_injection.md)  
6.  [Criptografía débil](./06_cryptography.md)  

## Resumen

La mayoría de las vulnerabilidades identificadas presentan un nivel de severidad alto o crítico, lo que implica un riesgo significativo en un entorno real.

Estas debilidades permiten desde accesos no autorizados hasta la ejecución remota de código en el sistema.
